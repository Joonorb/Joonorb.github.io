Projects

Flag Guesser

After a period of obsession with an online game called GeoGuesser, where users are placed into a location via Google Maps and have to guess their location, I came up with the idea of Flag Guesser. I wanted to improve my ability to recognize country flags, which I struggled with in the game. Flag Guesser is an engaging web-based quiz game that challenges users to identify countries based on their flags, utilizing HTML, CSS, JavaScript, and the REST Countries API. Implemented features include flag displays, real-time scoring, medals based on final scores, and interactive hints.

ProcrastiMate

A productivity application designed to combat procrastination by managing and scheduling computer access. Developed using Python with Tkinter for the UI, the app integrates task scheduling and distraction-blocking features. Users create "tasks," specifying time periods during which certain websites and applications on their PC will be restricted. For example, the user might have a task called "coding session," where VS Code will automatically open, and websites like YouTube and applications like Steam will be closed and blocked until the task ends. Core functionalities were implemented using libraries such as schedule for task scheduling and plyer for notifications.
